## Getting Started

```bash
git clone https://github.com/dimas/dimas.com.git
cd dimas.com
```

Create a `.env` file similar to [`.env.example`](https://github.com/dimas/dimas.com/blob/main/.env.example).

Then install dependencies and run the development server:
```sh-session
pnpm install
pnpm dev
```
