This is my personal website and portfolio. It's built with Next.js, Tailwind CSS, and Contentlayer.

## Features

- **Next.js 13** with App Router
- **Tailwind CSS** for styling
- **Contentlayer** for content management
- **Vercel** for deployment

## Tech Stack

- Next.js 13
- TypeScript
- Tailwind CSS
- Contentlayer
- Vercel

## Getting Started

```bash
git clone https://github.com/dimas/dimas.com.git
cd dimas.com
npm install
npm run dev
```

Visit [dimas.com](https://dimas.com) to see it in action!
