A simple access control library for Node.js that provides role-based access control (RBAC) functionality.

## Features

- Role-based access control
- Permission management
- Easy to integrate
- TypeScript support

## Installation

```bash
npm i @dimas/access
```

## Usage

```typescript
import { AccessControl, Role } from "@dimas/access";

const ac = new AccessControl();

// Define roles and permissions
ac.grant('user').read('post');
ac.grant('admin').extend('user').create('post');

// Check permissions
const canRead = ac.can('user').read('post').granted;
```
